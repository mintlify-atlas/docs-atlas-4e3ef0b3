SatDump is available for Android and supports nearly all of its features on mobile devices. You can install it from F-Droid or download an APK directly from the GitHub Releases page.

## Installation

The preferred source for SatDump on Android is **F-Droid**, which provides verified builds and automatic updates.

[Install from F-Droid](https://f-droid.org/packages/org.satdump.SatDump)

If F-Droid is not an option, APKs are also available on the [GitHub Releases page](https://github.com/altillimity/SatDump/releases).

## Supported SDR devices

The following SDR devices are supported for live processing on Android:

- RTL-SDR
- Airspy
- AirspyHF
- LimeSDR Mini
- HackRF

<Note>
  Not all SDR devices are supported on Android. Some hardware may have limitations that prevent it from working, depending on the device's USB implementation and driver compatibility.
</Note>
